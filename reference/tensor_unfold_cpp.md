# Tensor Unfolding (Matricization) for Synthetic Interventions

Tensor Unfolding (Matricization) for Synthetic Interventions

## Usage

``` r
tensor_unfold_cpp(T_cube, mode)
```

## Arguments

- T_cube:

  A 3D array (cube) of dimensions (n1, n2, n3)

- mode:

  The mode to unfold along (1, 2, or 3)
